Ciao! This is Vera Rilossi, I recently graduated in Computer Programming and Design, magna cum laude in New York City.

I like to always push myself to ensure that my designs are the best I could possibly make with my current skillset. I never want to stop leaning and growing.

I've coded this responsive portfolio using HTML, CSS and JavaScript.
